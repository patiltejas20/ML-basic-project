# ML-basic-project

# Summary:
Data Analysis Key Findings
A synthetic dataset was generated with 1000 samples, 20 features (15 informative, 5 redundant), and 2 classes.
The numerical features of the synthetic data were successfully scaled using StandardScaler.
The scaled data was split into training and testing sets, with 80% for training and 20% for testing.
A Logistic Regression model was trained on the training data.
The trained Logistic Regression model achieved an accuracy of 0.825 on the test data.
